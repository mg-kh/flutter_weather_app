# Weather api app

App build with flutter.

--- 

## About App

* Explore weather data around the world.
* Can explore weather forecast for 7 days.
* Use open street map.

## How to use
``git clone https://github.com/mg-kh/flutter_weather_app.git``

``flutter pub get``

### Simple!
